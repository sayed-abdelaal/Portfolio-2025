Hiring UI Designer

Freelance UI Designer – Web Platform Education
Start date: ASAP
Commitment: 20–32 hours/week, 1 Month, Flexible Hours.
Location: Remote (Egypt based).

I'm looking for a freelance UI Designer to partner up with me on an education web platform. 
This role is perfect for someone who understands the nuances of designing for learners and educators and has a portfolio that reflects thoughtful, user-focused design for web platforms.

What You'll Do:

- Translate product requirements into intuitive, responsive interface designs.
- Design clean, user-friendly UI components for dashboards, interactive tools, and content displays.
- Work closely with me to ensure design feasibility and consistency.
- Apply accessibility best practices (WCAG 2.1+) to every part of the interface.
- Create and maintain design style guide, components library and UI documentation.
- Iterate on feedback quickly without compromising design quality.
- Deliver Figma files ready for handoff, with clear documentation for developers.

Your Profile:

- UI design experience, with at least 2 years working on web platforms.
- Strong portfolio showing web-based UI work, preferably in education.
- Expert-level skills in Figma.
- Understanding of responsive and mobile-first design principles.
- Familiarity with WCAG guidelines and accessible UI patterns.
- Ability to work independently and communicate clearly in async and remote-first environments.
- Experience collaborating with UX designers.

If interested, please send me a short note with:
A link to your portfolio.
Your availability and hourly rate.
